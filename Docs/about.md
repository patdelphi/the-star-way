# GitHub 仓库 About 区域文案

> 用于 GitHub 仓库右侧 About 区域。
> 打开 https://github.com/patdelphi/the-star-way → 右侧 About ⚙️ → Description 填入下方文案。

## 英文卖点（推荐主用）

```
Turn GitHub Stars into an AI-powered developer interest map and learning path.
```

## Website

```
https://starway.patdelphi.xyz
```

## 备选文案

短版（≤70 字符）：
```
Analyze GitHub Stars → developer tech profile + learning path
```

中文版：
```
把 GitHub 星标仓库变成开发者技术画像和学习路径。AI 增强分析，支持本地 SQLite 与 Cloudflare 双部署。
```

## 设置方式

1. 打开 https://github.com/patdelphi/the-star-way
2. 点击右侧 About 区域的 ⚙️ 齿轮图标
3. Description 填入英文卖点
4. Website 填入 demo 地址
5. 勾选 "Releases" 和 "Packages"（如需要）
6. 点击 Save changes
