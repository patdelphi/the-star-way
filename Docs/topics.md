## 设置方式

1. 打开 https://github.com/patdelphi/the-star-way
2. 点击右侧 About 区域的 ⚙️ 齿轮图标
3. 在 Topics 输入框逐个粘贴上述关键词（回车确认每一个）
4. 点击 Save changes
