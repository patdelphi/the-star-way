## Release notes

```markdown
# the-star-way v0.1.0

Turn GitHub Stars into a developer interest map and AI-powered learning path.

## Highlights

- **Star Sync** — Sync starred repositories from any GitHub user, with incremental updates and removed-star tracking
- **Smart Classification** — Auto-categorize repos by topics, languages, and descriptions (60+ categories)
- **Star DNA** — Generate a developer tech profile from starred repos (AI-powered)
- **Learning Path** — Personalized learning path recommendations (AI-powered)
- **README Summary** — Intelligent summary of repo READMEs (AI-powered)
- **Hidden Gems & Sleeping Stars** — Surface overlooked low-star projects and dormant repos
- **Multi-dimensional Filtering** — Filter by language, tag, keyword; sort and paginate
- **Statistical Analysis** — Language distribution, topic clustering, license compliance, star timeline trends
- **Data Export** — Export stars as CSV / JSON / Markdown / HTML
- **Dual Deployment** — Run locally with Node.js + SQLite, or deploy on Cloudflare Workers + D1 + Pages
- **Bilingual UI** — Chinese / English switch
- **Demo Mode** — Built-in real starred repo data, works without backend

## Tech Stack

React 19 · Vite · TypeScript · Tailwind CSS 4 · Radix UI · recharts · Cloudflare Workers · D1 · SQLite

## 设置方式

1. 打开 https://github.com/patdelphi/the-star-way/releases/new
2. Choose a tag → 输入 `v0.1.0` → Create new tag
3. Release title 填 `v0.1.0 - First public release`
4. 复制上方 Release notes 内容到描述框
5. 勾选 "Set as the latest release"
6. 点击 Publish release
